# 広島市タクシーダッシュボード

広島市内でタクシー乗務前に確認したい情報を、スマートフォンから素早く開けるようにまとめた個人用ダッシュボードです。

道路交通情報、公共交通機関の運行情報、空港、スポーツ・イベント情報など、乗務前の確認に役立つ公式サイトへのリンクを整理しています。

## 公開ページ

- 手動リンク版  
  https://toyoji-afk.github.io/hiroshima-taxi-dashboard/

- 自動巡回版  
  https://toyoji-afk.github.io/hiroshima-taxi-dashboard/auto/

## 主な内容

- 道路・規制情報
- 鉄道・バス運行情報
- 広島空港フライト情報
- カープ、サンフレッチェ、主要ホール・イベント情報
- タクシー業務に関連する公式サイトへのリンク
- 乗務前メモ欄

## 自動巡回版について

自動巡回版では、GitHub Actionsを使って一部の公式サイトを定期的に確認し、乗務前に見ておきたい情報を簡易メモとして表示しています。

ただし、巡回結果はあくまで補助情報です。実際の運行状況、道路状況、イベント開催状況などは、必ず各公式サイトで確認してください。

## 注意事項

このダッシュボードは個人が作成・管理している非公式ツールです。

掲載している各リンク先の情報は、それぞれの公式サイト・運営者に帰属します。本ダッシュボードは、各情報を保証するものではありません。

情報の取得失敗、更新遅れ、サイト構成変更、通信環境などにより、表示内容が実際の状況と異なる場合があります。

タクシー乗務、運行判断、道路選択、接客対応などの最終判断は、必ず各公式情報、会社の指示、法令、現場状況に基づいて行ってください。
